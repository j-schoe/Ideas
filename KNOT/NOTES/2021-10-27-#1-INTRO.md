# (K)not

Something like docker or node just fully based
on webassembly.

```
$ knot <command> [<args>...]
```

## Exporting

```
$ knot say Hello World
$ knot export add say --as say
$ say Hello World
```

```
$KNOT_HOME/bin/say
$KNOT_HOME/bin/say.bat
```

## Installing

```
$ # add repository
$ knot repos add pit://say.example/
$ # fetch image
$ knot image add say:latest
$ # create instance
$ knot init say:latest -n say
```
